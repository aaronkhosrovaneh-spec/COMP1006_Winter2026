1. What is the purpose of the $_FILES superglobal? - It is an associative array that stores information about uploaded items, such as name, type, size and temporary storage location.
2. Why does a form need special settings? - To upload files, the form must use the POST method and include enctype="multipart/form-data" so the server knows how to handle the binary data.
3. What function moves uploaded files? - The move_uploaded_file() function.
4. Why is it important to control where files are uploaded? - For security and organization, ensuring files are stored in a designated, accessible directory like uploads.
